# gaofang_wyykt

#### 项目介绍
 Vue-wyclass 仿网易云课堂App
功能实现：
首页
1、搜索框
2、tabbar切换
3、swiper滑动，切换页面
4、首页各个界面
分类
1、实现切换分类模块
2、右侧菜单
我的学习
1、判断登录状态
2、用户所学课程展示

个人
1、账户的登录
2、设置界面 退出当前账号

搜索界面
1、根据用户输入查找所有课程中符合要求的课程并显示

课程详细界面
1、通过router传参查找课程
1、根据用户是否拥有选择播放视频权限
2、课程介绍界面
